Projects I've built.

I

The Potter's House

Inventory & Procurement Management System.
(Python (Django), MySQL, HTML/CSS, Bootstrap CSS, JavaScript (Chart.js))

A real business needed a better way to manage their products and orders — so I built them one. Before my system, the staff tracked everything on paper, which led to costly errors. I created a clean web dashboard where they can check stock levels, manage purchase requests, approve orders, and calculate product costs automatically. I also added visual charts so the owners can instantly see sales trends and make smarter decisions. This project saved the business hours of manual work every week.

II

KickX E-commerce

Sneaker Retail Platform with Recommendation Engine.
(Python (Flask), Supabase, HTML/CSS, Bootstrap 5, JavaScript (PayPal SDK), Data Analysis (Pandas/NumPy))

I built a complete online store from the ground up where users can browse sneakers, add them to their cart, save favorites to a wishlist, and pay safely through PayPal. What makes this store special is the smart recommendation system I created — it tracks what's popular and suggests the right products to each user, which helps drive more sales. The store owner also gets an admin panel to manage products, view orders, and track everything in one place. This project shows I can build e-commerce solutions that actually sell.

III

SafeBite

Android Application for Crime-Safe Travel with AR Navigation. (React Native, Python (FastAPI), Unity, Firebase, Mapbox)

SafeBite is my Capstone Project that was recognized as the Best Research Paper. We developed it because safety is a real problem for people traveling in unfamiliar areas. Our team built SafeBite to solve that — it uses crime data to find the safest walking or driving routes to nearby restaurants. The app's best feature is its AR navigation, which overlays turn-by-turn directions on your phone's camera so you never get lost. I was responsible for building the backend server that handles all the map data, user routes, and safety calculations quickly and securely. This project proves I can work with complex data and deliver apps that truly help people.

IV

LoKal AI Assistant

Offline Educational AI Learning Assistant.
(Python (Django), JavaScript, HTML/CSS, SQLite, Ollama, Vosk, Raspberry Pi 4B)

Many students in the Philippines don't have internet access, but they still deserve great learning tools. I built LoKal AI to solve this — it runs entirely offline on a small Raspberry Pi computer and hosts a smart AI chatbot that answers study questions anytime. The system even creates its own Wi-Fi hotspot automatically, so students just connect their phones and start learning with zero setup. This project shows I can build complete systems that work in the most challenging environments, not just perfect lab conditions.

V

PHQuest

Philippine Discovery Quiz Game.
(React Native, Python (FastAPI), SQLite)

I wanted to help people learn about Philippine history and culture in a way that's actually fun. PHQuest is an interactive mobile game where players complete quests and quizzes about the country's geography, history, and traditions. I built a real-time global leaderboard so players can compete with each other, plus a detailed performance tracker that shows how much they've learned. The full backend server handles all user data, scoring, and rankings. It proves I can build engaging mobile experiences with solid backend systems.

VI

Garbage Bot Picker

Gesture-Controlled Robotic Vehicle.
(Python (Flask, OpenCV), NodeMCU, MPU6050, Raspberry Pi (GPIO), ESP32-CAM, TCP/IP Sockets)

This project combined web development with robotics. I programmed a vehicle that responds to hand gestures in real time — tilt your hand forward and it drives forward, rotate it and the arm grabs objects. It streams live video from an onboard camera so you can see exactly what the robot sees. I built the Flask web server that connects the sensors to the motors and handles the video feed. This project proves I can work beyond just websites and bring code into the physical world when needed.
